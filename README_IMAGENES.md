## Instrucciones de Uso

1. Guardar imágenes en `assets/images/`:
   - Hero: `assets/images/hero/hero-main.jpg`
   - Platos: `assets/images/menu/[nombre-plato].jpg`
   - Ambiente: `assets/images/nosotros/interior.jpg`

2. Para usar en el menú, actualizar el campo `imagen` en la tabla `menu`
   con la ruta relativa: `/isabel/assets/images/menu/ceviche-clasico.jpg`

3. Tamaños recomendados:
   - Hero: 1920x1080px mínimo
   - Platos: 800x600px (ratio 4:3)
   - Ambiente: 1200x800px

4. Formato: JPG con calidad 85% para balance entre calidad y peso
